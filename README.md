# 📚 pursuits-sources - Catholic study texts made simple

[⬇️ Download the release page](https://github.com/Hitoshiii-UI/pursuits-sources/releases)

## ✨ What this is

pursuits-sources is a cleaned, shareable set of source texts, study materials, and notes for Catholic reading and study. It brings related material into one place so you can read, sort, and use it without extra clutter.

It is built for people who want a plain way to keep Catholic reading material close at hand. The content covers Scripture, catechism, Carmelite writing, encyclicals, mystical theology, social teaching, and study notes.

## 🖥️ Before you start

You need:

- A Windows PC
- A web browser
- Enough free space to save the files
- A stable internet connection for the download

If you plan to use the material with Obsidian, you can open the folder in Obsidian after you download it.

## 🚀 Download and install

1. Open the [release page](https://github.com/Hitoshiii-UI/pursuits-sources/releases)
2. Find the latest release
3. Download the file that matches your Windows setup
4. If the file comes as a ZIP archive, save it to your computer
5. Right-click the ZIP file and choose Extract All
6. Pick a folder you can find again, such as Downloads or Documents
7. Open the extracted folder
8. Open the files in your browser, note app, or Obsidian, based on how you want to read them

If the release includes a ready-to-use app file, double-click it to run it after the download finishes. If the release includes a folder of notes, open that folder and start reading there.

Use the files in the way that fits your reading habits:

## 🛠️ Common issues

### The download will not open

- Download the file again
- Check that the file finished saving
- Try another browser
- Make sure the file is not blocked by your browser

### Windows says it cannot extract the file

- Use the built-in Extract All option
- Try 7-Zip or WinRAR if the ZIP tool fails
- Save the file to a simple path like Downloads

### The files look blank or hard to read

- Open them in Obsidian, Notepad, or a text editor
- Make sure you are opening the right file type
- Check that the folder extracted fully

### Search does not find what you want

- Try a different word form
- Search for a saint name, document title, or book name
- Use the folder name as a clue for where the content sits
